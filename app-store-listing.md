# ExpatHub - App Store Listing

> **ASO note:** Apple ranks on **app name + subtitle + keyword field** only.
> Promotional text and description are not indexed (they only affect
> conversion). The keyword field below intentionally omits any word that
> already appears in the app name or subtitle — Apple combines tokens across
> all three fields, so repeating "expat", "relocation", "visa", "residency",
> "move", "moving", or "planner" in the keyword field would waste characters.
> Phrases are also broken into single comma-separated tokens (no spaces)
> because Apple auto-recombines them, which yields more ranking combinations
> per character.

## ASO setup checklist (App Store Connect)

- [ ] Update **App Name**, **Subtitle**, and **Keywords** in App Store Connect for the en-US locale (changes require a new version submission).
- [ ] Add **English (UK)** as a second localization and paste the same name / subtitle / keywords. This gives a second independent 100-char keyword field that Apple also indexes — biggest free ASO lift for an English-only app.
- [ ] Add **English (Australia)** as a third localization for the same reason.
- [ ] Confirm primary category is **Travel** (or test **Lifestyle** as a secondary).
- [ ] Set up **In-App Events** in App Store Connect for any new country launch, quiz refresh, or feature release — events are indexed and surface on the product page.
- [ ] Create at least one **Custom Product Page** with screenshots and promo text tailored to a single audience (e.g., "digital nomad" or "retire abroad") so paid acquisition can deep-link to it without changing the default listing.
- [ ] Audit screenshots: front three should each carry one large benefit headline ("Find your country in 2 minutes" / "Compare 11 destinations on 14 factors" / "Plan your move step by step") rather than feature labels on device frames.
